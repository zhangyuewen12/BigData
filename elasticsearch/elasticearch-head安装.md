

软件包地址 ：https://github.com/mobz/elasticsearch-head

## 下载运行

- `git clone git://github.com/mobz/elasticsearch-head.git`
- `cd elasticsearch-head`
- `npm install`
- `npm run start`



## 解决跨域问题  修改elasticsearch.yml

http.cors.enabled: true

http.cors.allow-origin: "*"
